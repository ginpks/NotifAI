notifications_filters Up

method (optional)
String is blank, do not filter by method
sent
Boolean true means sent by user, false means user is receiving notifications
type (optional)
String if empty, do not filter by type
archived (optional)
Boolean if true, leave only archived notifications, if false or undefined, do not filter by archived

String filter from the searching feature of user inbox, filters title, description, and users involved in notifications for matching text
